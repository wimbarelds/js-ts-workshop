# Callbacks

## Wat is een callback?

Een **callback** is een functie die als parameter wordt doorgegeven aan een andere functie.

```js
function plus(x, y) {
  return x + y;
}
function times(x, y) {
  return x * y;
}

function handleNumbers(x, y, callback) {
  const result = callback(x, y);
  console.log(result);
}

// Gebruik de callback
handleNumbers(5, 7, times);
// Resultaat: 35
```

## Callbacks met arrow functies

Vaak wordt voor callbacks een inline arrow functie gebruikt. Ofwel:

```js
handleNumbers(3, 5, (x, y) => x + y);
// Resultaat: 8
```

## Array callbacks

JavaScript heeft een flink aantal array-functies die een callback gebruiken.
Deze functies krijgen de waardes uit een array 1 voor 1.

Bijvoorbeeld, stel:

```js
// Deze functie returned `true` als de waarde eem nummer is, anders false
function isNumber(value) {
  return typeof value === 'number';
}

const myArray = [1, 'hallo', false, 2, true, 'sopra', 3, 'test'];
const myArrayNumers = myArray.filter(isNumber);

console.log(myArrayNumbers);
// Resultaat: [1, 2, 3]
```

### `Array.filter()`

Zoals hierboven, de filter methode geeft een nieuwe array terug op basis van het origineel, maar
alleen met de waardes waarbij de callback functie `true` terug geeft.

```js
const getallen = [1, 2, 3, 4, 5, 6];

const evenGetallen = getallen.filter((getal) => getal % 2 === 0);
console.log(evenGetallen); // [2, 4, 6]
```

### Array.find()

Vindt het eerste element dat voldoet aan de conditie (waar de functie `truez returned)

```js
const personen = [
  { naam: 'Alice', leeftijd: 25 },
  { naam: 'Bob', leeftijd: 30 },
  { naam: 'Charlie', leeftijd: 35 },
];

const oudere = personen.find((persoon) => persoon.leeftijd > 28);
console.log(oudere); // { naam: 'Bob', leeftijd: 30 }
```


### Array.map()

Maakt een nieuwe array waarin de nieuwe waardes afgeleid zijn van de oude waardes uit de originele
array.

```js
const getallen = [1, 2, 3, 4];

const dubbels = getallen.map((getal) => getal * 2);
console.log(dubbels); // [2, 4, 6, 8]
```
