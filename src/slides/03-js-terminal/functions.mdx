# Functies

## Wat is een functie?

Een functie is een **herbruikbaar blok code** dat je kunt aanroepen. Het wordt ook gebruikt om
code die bij elkaar hoort te verzamelen, dat helpt bij het makkelijker leesbaar maken van code.

```js
function sayHi() {
  console.log('Hallo!');
}

// Functie aanroepen
sayHi(); // "Hallo!"
```

## Parameters

Functies kunnen **input** ontvangen via parameters.

```js
function sayHi(naam) {
  console.log(`Hallo ${naam}!`);
}

sayHi('Alice'); // "Hallo Alice!"
sayHi('Bob');   // "Hallo Bob!"
```

## Return waarden

Functies kunnen een **waarde teruggeven**.

```js
function optellen(a, b) {
  return a + b;
}

const resultaat = optellen(3, 5);
console.log(resultaat); // 8
```

## Arrow functies

Een **kortere manier** om functies te schrijven.

```js
// Traditionele functie
function kwadraat(x) {
  return x * x;
}

// Arrow functie (gelijk)
const kwadraat = (x) => {
  return x * x;
};

// Extra kort (impliciete return)
const kwadraat = (x) => x * x;
```

## Wanneer welke gebruiken?

```js
// Arrow functies voor korte, simpele functies en bij nested functies (een functie in een functie)
const dubbel = (x) => x * 2;
const isEven = (x) => x % 2 === 0;

// Traditionele functies voor complexere logica en top-level functies
function berekenLeeftijd(geboortejaar) {
  const huidigJaar = new Date().getFullYear();
  return huidigJaar - geboortejaar;
}
```
